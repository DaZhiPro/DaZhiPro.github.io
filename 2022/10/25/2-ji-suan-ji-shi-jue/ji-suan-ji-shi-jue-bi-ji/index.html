先验 后验 和 似然函数的关系

常见的概率分布模型

概率分布就是一个公式，给定X，就能计算出在这一点X的概率（或者连续随机变量的话就是这一点的概率密度）

伯努利分布
描述一个像素的灰度值大于或者小于128的概率

P(x = 0) = 1 − λ   (0:灰度值小于128)P\left( x\ =\ 0 \right) \ =\ 1\ -\ \lambda \ \ \ \left( 0:\text{灰度值小于}128 \right) P(x = 0) = 1 − λ   (0:灰度值小于128)

P(x = 1)  = λ (1:灰度值大于128)P\left( x\ =\ 1 \right) \ \ =\ \lambda \ \left( 1:\text{灰度值大于}128 \right) P(x = 1)  = λ (1:灰度值大于128)

或者统一为\text{或者统一为} 或者统一为

P(x) = λx ∗ (1−λ)1−xP\left( x \right) \ =\ \lambda ^x\ *\ \left( 1-\lambda \right) ^{1-x} P(x) = λx ∗ (1−λ)1−x

贝塔分布
一个连续的分布，
由单变量λ定义的连续分布，λ=[0,1],适合表示伯努利分布的参数λ的不确定性\text{由单变量}\lambda \text{定义的连续分布，}\lambda =\left[ 0,1 \right] ,\text{适合表示伯努利分布的参数}\lambda \text{的不确定性}由单变量λ定义的连续分布，λ=[0,1],适合表示伯努利分布的参数λ的不确定性

分类分布
一个离散的分布，用来表示kkk个可能的结果的概率λk\lambda _kλk​
狄利克雷分布
一个连续的分布，用于表示kkk个连续值(λ1...K\lambda _{1...K}λ1...K​)的取值情况，其中∑k=1Kλk=1\sum_{k=1}^K{\lambda _k}=1∑k=1K​λk​=1，适合表示分类分布参数λk\lambda _kλk​的分布

拟合概率模型的方法——极大似然、最大后验、贝叶斯方法

基本概念
先验 P(θ):不考虑样本，这个分布的参数应该满足什么样的分布 \text{先验\\ }P\left( \theta \right) :\text{不考虑样本，这个分布的参数应该满足什么样的分布} 先验 P(θ):不考虑样本，这个分布的参数应该满足什么样的分布
后验 P(θ∣x):在已知样本x的结果的前提下，分布的参数是θ的概率是多少 \text{后验\\ }P\left( \theta |x \right) :\text{在已知样本}x\text{的结果的前提下，分布的参数是}\theta \text{的概率是多少} 后验 P(θ∣x):在已知样本x的结果的前提下，分布的参数是θ的概率是多少
似然 P(x∣θ): 好像就是实际的分布，就是说实际的分布的参数假如说是θ 的话，那么x的概率是啥 \text{似然\\ }P\left( x|\theta \right) :\ \text{好像就是实际的分布，就是说实际的分布的参数假如说是}\theta \ \text{的话，那么}x\text{的概率是啥} 似然 P(x∣θ): 好像就是实际的分布，就是说实际的分布的参数假如说是θ 的话，那么x的概率是啥

最大似然拟合概率模型 ML

顾名思义就是说最大化 P(x∣θ)=P(x1..n∣θ)=∏i=1nP(xi∣θ)P\left( x|\theta \right) =P\left( x_{1..n}|\theta \right) =\prod_{i=1}^n{P\left( x_i|\theta \right)}P(x∣θ)=P(x1..n​∣θ)=∏i=1n​P(xi​∣θ)，求解其中的未知量θ\thetaθ

arg max⁡θP(x∣θ)\argmax_{\theta}P\left( x|\theta \right) θargmax​P(x∣θ)

=arg max⁡θP(x1..n∣θ)=\argmax_{\theta}P\left( x_{1..n}|\theta \right) =θargmax​P(x1..n​∣θ)

=arg max⁡θ∏inP(xi∣θ)=\argmax_{\theta}\prod_i^n{P\left( x_i|\theta \right)} =θargmax​i∏n​P(xi​∣θ)

=arg max⁡θ ∑log⁡(P(xi∣θ))=\argmax_{\theta}\ \sum{\log \left( P\left( x_i|\theta \right) \right)} =θargmax​ ∑log(P(xi​∣θ))

最大后验拟合概率模型 MAP

顾名思义就是最大化后验

arg max⁡θP(θ∣x)\argmax_{\theta}P\left( \theta |x \right) θargmax​P(θ∣x)

=arg max⁡θP(x∣θ) × P(θ)P(x)=\argmax_{\theta}\frac{P\left( x|\theta \right) \ \times \ P\left( \theta \right)}{P\left( x \right)} =θargmax​P(x)P(x∣θ) × P(θ)​

=arg max⁡θP(x∣θ) ∗ P(θ)=\argmax_{\theta}P\left( x|\theta \right) \ *\ P\left( \theta \right) =θargmax​P(x∣θ) ∗ P(θ)

他有个好处就是可以利用到先验的知识，不像极大似然估计只知道用样本的数据进行拟合，这个还考虑到了先验的概率，但是先验的参数怎么确定呢，有利也有弊，后验方式是先验和极大似然方式的加权平均

贝叶斯方式

其实我们的最终目的不是求参数θ\thetaθ,而是给定一个新数据x∗x^*x∗要计算P(x∗∣X1..I)P(x^*|X _{1..I})P(x∗∣X1..I​)的概率 (X1..IX _{1..I}X1..I​ 表示已有的抽样的数据) ，之前都是找出一个概率最大的θ\thetaθ,其实这个θ\thetaθ也满足一个分布，我们直接对他进行每个概率都给他乘上，然后加权一下就好了

P(x∗∣X1..I) = ∫P(x∗∣θ) ∗ P(θ∣x) dθP\left( x^*|X _{1..I} \right) \ =\ \int{P\left( x^*|\theta \right) \ *\ P\left( \theta |x \right) \ d\theta} P(x∗∣X1..I​) = ∫P(x∗∣θ) ∗ P(θ∣x) dθ

贝叶斯方式去除了计算参数θ\thetaθ的误差，更加合理

生成模型和判别模型

判别模型： 比较简单直接计算 P(w∣x)这里的w叫做全局状态(我理解就是y)P(w|x) 这里的w叫做全局状态(我理解就是y)P(w∣x)这里的w叫做全局状态(我理解就是y) 大概就是先估计一下w的分布，比如说在例子通过轮胎像素的个数，来估计车身的大小的样例中，w估计就是一元正态分布
P(w∣x,θ)=Normw[a+bx,σ2]这里的θ就是一些需要确定得超参数，包括a,b,σ,所以只要求出θ的估计值就可以进行推理(我立即的是预测)了。确定超参数有好多算法，比如可以用MAP算法求arg max⁡θP(x∣θ) ∗ P(θ)P(w|x,\theta) = Norm_w[a + bx,\sigma^2] 这里的\theta就是一些需要确定得超参数，包括a,b,\sigma,所以只要求出\theta的估计值就可以进行推理(我立即的是预测)了。\\ 确定超参数有好多算法，比如可以用MAP算法求\argmax_{\theta}P\left( x|\theta \right) \ *\ P\left( \theta \right)P(w∣x,θ)=Normw​[a+bx,σ2]这里的θ就是一些需要确定得超参数，包括a,b,σ,所以只要求出θ的估计值就可以进行推理(我立即的是预测)了。确定超参数有好多算法，比如可以用MAP算法求argmaxθ​P(x∣θ) ∗ P(θ)

生成模型：假如说是一个分类问题，就拟合分类为0的数据的模型，然后拟合分类为1的数据的模型，然后对于一个给定的数据，带入到两个模型，看看哪个概率打， 就属于哪个【这是w为离散的情况，好理解些】
【w连续的情况】 要计算的是P(x∣w,θ)就是说在每一个全局状态w下，存在x这种数据的概率,通过样本xi,wi可以计算出参数θ从而求得P(x∣w)//这种情况需要先预估w的分布，然后用现有的wi样本去拟合这个分布，得到P(w)这样就可以利用贝叶斯公式计算出后验P(w∣x)=P(x∣w)∗p(w)/p(x)要计算的是 P(x|w,\theta) 就是说在每一个全局状态w下，存在x这种数据的概率,通过样本{x_i,w_i} 可以计算出参数\theta从而求得P(x|w)// 这种情况需要先预估w的分布，然后用现有的w_i样本去拟合这个分布，得到P(w) 这样就可以利用贝叶斯公式计算出后验P(w|x) = P(x|w) * p(w) / p(x)要计算的是P(x∣w,θ)就是说在每一个全局状态w下，存在x这种数据的概率,通过样本xi​,wi​可以计算出参数θ从而求得P(x∣w)//这种情况需要先预估w的分布，然后用现有的wi​样本去拟合这个分布，得到P(w)这样就可以利用贝叶斯公式计算出后验P(w∣x)=P(x∣w)∗p(w)/p(x)
